### Including Your Name

To be included in this file:

1. **Code Contributions**:
   - Submit a merged pull request
   - Add your GitHub username in PR
   - You'll be added in next release

2. **Bug Reports**:
   - Report a valid bug that gets fixed
   - We'll credit you in changelog and here

   - Report a security vulnerability responsibly
   - We'll credit you with your permission

   - Submit documentation improvements

## ğŸ“ˆ How to View All Contributors

To see all contributors programmatically:

```bash
# View all commit authors
git log --format='%aN' | sort -u

# View contribution statistics
git shortlog -sn --all --no-merges

# View contributors by lines changed
git log --shortstat --author="YourName" | grep -E "fil(e|es) changed" | \
  awk '{files+=$1; inserted+=$4; deleted+=$6} END \
  {print "Files: ", files, "Insertions: ", inserted, "Deletions: ", deleted}'
```
